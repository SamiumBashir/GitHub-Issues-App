# 1️⃣ Difference between var, let, const

var → old way to declare variable

let → can change value

const → value cannot be changed

# 2️⃣ What is the spread operator (...)

It copies or expands arrays or objects.

# 3️⃣ Difference between map(), filter(), forEach()

map() → changes each item and returns new array

filter() → selects items based on condition

forEach() → just loops through items

# 4️⃣ What is an arrow function

A short way to write a function.
const add = (a,b) => a + b;

# 5️⃣ What are template literals

A way to write strings with variables inside using backticks `.
